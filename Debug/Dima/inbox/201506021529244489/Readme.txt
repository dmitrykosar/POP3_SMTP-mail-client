1. Особенности программы и минимальные требования для ее запуска

- в программе предусмотрены четыре тренировочных режима решения билетов:
по главам ПДД (вопросы распределены по главам ПДД), по тематическим, комплексным
и случайным билетам. В процессе тренировки при втором неправильном ответе
"подсказывается" правильный и дается ссылка на соответствующий пункт ПДД;

- режим контроля осуществляется в четырех видах: по тематическим билетам, по
строго фиксированным билетам, по случайным билетам (вопросы для билета 
подбираются случайным образом) и по случайным билетам "x10". Допускается один
неправильный ответ;

- для тематического билета без номера вопросы подбираются случайным образом из
выбранной темы (аналог зачета);

- во всех режимах контроля  используется функция "плавающий ответ" (номер 
правильного ответа для одной и той же задачи будет каждый раз разным). Таким 
образом предотвращается неосознанное заучивание правильных ответов, повышаются 
внимательность и качество подготовки обучаемого;

- во всех режимах контроля  по завершению экзамена выдается список  допущенных 
ошибок с возможностью разбора тех задач, на которые были даны  неправильные 
ответы;

- в режиме "Контроль по случайному билету x10" результаты будут выданы только
после решения всего билета;

- в программе имеются встроенные Правила дорожного движения. В процессе решения 
билета в режиме тренировки при нажатии на клавишу [F1] на экран выводятся 
положения Правил,  касающиеся поставленного вопроса.  В режимах контроля  это 
можно сделать только после завершения решения билета при просмотре допущенных 
ошибок;

- во всех тренировочных режимах во время решения  билета, нажав на клавишу [F2],
Вы можете получить подробные пояснения к поставленному вопросу. В режимах 
контроля это можно сделать только после завершения решения билета при просмотре
допущенных ошибок;

- программа  поддерживает два языка: русский и белорусский;

- программу можно активировать и запускать на компьютере, отвечающем необходимым
минимальным системным требованиям.

Минимальные системные требования для запуска программы:  процессор Intel или
AMD x86/x64 с частотой от 2 GHz, видеоадаптер с поддержкой цветовой палитры 16
бит и выше, клавиатура, мышь, операционная система Microsoft Windows XP/Vista/7/8
с последним пакетом обновлений, учетная запись с правами администратора.
Для активации программы компьютер должен иметь доступ к сети Интернет. Выполнять
активацию программы необходимо на территории Республики Беларусь.
Внимание! Программа не работает под эмулятором компьютера (виртуальной машиной).


2. Запуск и работа с учебной программой ПДД

Найдите на рабочем столе ярлык "Учебная программа ПДД - 2015" и запустите его.
При первом запуске программы требуется произвести ее активацию. Введите в окне
активации серийный номер, приобретенный Вами ранее и нажмите кнопку "Далее".

Внимание! После активации программы нельзя изменять программно-аппаратную
конфигурацию компьютера (центральный процессор, объем памяти, системную плату
и т.п.). В случае значительного изменения программно-аппаратной конфигурации
компьютера программа не запустится и повторная активация будет невозможна.

Когда программа запустится, Вы увидите меню  с 8-ю режимами работы: тренировка 
по главам ПДД, тренировка и контроль по тематическим билетам, тренировка и 
контроль по строго фиксированным билетам, тренировка и контроль по случайным 
билетам и контроль по случайным билетам "x10".

Режимы тематических билетов включает в себя семь частей, вопросы в которых 
распределены по ниже перечисленным главам ПДД:
1) Главы 1-6;
2) Дорожные знаки и разметка. Приложения 2-3;
3) Главы 7, 13. Приложение 1;
4) Главы 8-12 и 19;
5) Главы 14-18, 20-25;
6) Глава 26. Приложение 4;
7) Ответственность. Безопасность. Медицина.

Тематический билет можно выбрать по номеру или сгенерировать случайный. Для того,
чтобы сгенерировать случайный тематический билет, выберите нужную тему и дважды
нажмите на клавишу [Enter].

В режиме тренировки или контроля по строго фиксированным билетам Вы выбираете 
номер билета и пробуете ответить на предложенные вопросы. Вопросы строго 
привязаны к номеру билета.  В режиме тренировки или контроля  по случайным 
билетам вопросы  компонуются в  билет случайным образом в определенной 
пропорции. Этот режим  аналогичен экзамену ГАИ. Во всех тренировочных режимах 
время ответа не ограничено и после второго неправильного ответа "подсказывается" 
правильный ответ и дается ссылка на соответствующий пункт ПДД.

Режим "Контроль по случайному билету x10" аналогичен теоретическому экзамену
автоучебной организации. После ввода ответа на вопрос задачи отобразится надпись
"ОТВЕТ ПРИНЯТ". Результаты будут выданы только после решения всего билета.

В режимах контроля время ограничено 15-ю минутами и допускается только один 
неправильный ответ.

Чтобы прервать решение задач билета и выйти в основное меню программы, нажмите 
клавишу [Esc].


3. Известные проблемы

- Если программа не запускается или возникает ошибка запуска, необходимо провести
полную антивирусную проверку операционной системы обновленным антивирусом
(Касперский, DrWeb или др.). После антивирусной проверки программу следует
переустановить;

- Если при нажатии на клавишу [F2] не отображается комментарий к задаче, 
отключите на клавиатуре функции офисных приложений (например, "Office Lock")
или, если ноутбук, используйте сочетание клавиш [Fn]+[F2]. Обратитесь
к документации Вашей клавиатуры или ноутбука. Клавиши [Z] и [X] дублируют [F1]
и [F2] соответственно;

- Для необновленной Windows XP возможно потребуется установить корневые 
сертификаты VeriSign. Если возникла проблема запуска программы, установите
сертификаты http://www.verisign.com/support/roots.zip или обновите Internet
Explorer: http://microsoft.com/rus/ie

При появлении других проблем закройте все приложения, обновите драйвера 
основных комплектующих Вашего компьютера,  установите программные "заплатки" и 
обновления Microsoft.

6) версия операционной системы;
